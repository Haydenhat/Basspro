Active Directory

Active directory will primarily be used to reset outfitter passwords, accounts will be accessed via searching their "FlexID".

The Organization tab can be accessed to view their job title and department, the account tab to view if the account is locked. The Member Of tab is used frequently to check certifications (accesses).


Microsoft Endpoint Manager

Microsoft Endpoint Managers primary use includes pushing software to outfitter computers along with remotely accessing outfitter computers in order to assist them in troubleshooting current issues.

Right clicking a computer can populate many options, some very useful ones being access to the C:, clearing caches, pinging the system and even opening Regedit.

IBMI (Cabelas I Series, Systems One - Four

Our IBMI (AS400 systems) house our .... and have a large amount of functions.

The most common functions you'll run into while using the AS400 system would be restarting rfguns, clearing out printer ques or restsrating them as well. There's also a password reset function (INZPWD) to reset outfitters passwords to their usernames. Releasing batch files will be semi-common.

SD Console

SD Console is another tool you can use in place of our Microsoft Endpoint Manager

The SD Console is quite useful, you can very quickly ping a device, open the CMD console on it from your workstation, run a system file check reboot/ shutdown the PC, check event viewer and even clean temporary files out all from the initial GUI.
